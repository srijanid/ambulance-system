How A* works — f(n) = g(n) + h(n)
g(n)
Actual cost from start to node n — accumulated edge weights along the best known path so far.
h(n)
Heuristic estimate of cost from n to goal. Uses Manhattan distance for grid graphs — admissible and consistent.
Priority queue
A min-heap ordered by f(n). Nodes with lower total estimated cost are expanded first, pruning unnecessary branches.

Dispatch algorithm logic
1. Emergency submitted with severity level (critical/high/medium/low)
2. Priority queue orders pending emergencies — critical first, then by creation time
3. A* calculates cost from each available ambulance to the emergency node
4. Nearest ambulance (lowest A* cost) is dispatched
5. Best hospital selected by A* distance from emergency scene
6. Ambulance marked dispatched until emergency resolved, then returned to available
